# Mokups (mockups)

Esta carpeta es solo para **mockups y experimentos de UI**.

## Reglas

1. **Excepción al design system:** aquí puedes usar Tailwind “crudo” (`text-white/50`, hex sueltos, etc.), otra librería, o componentes que aún no migraste a `src/theme/tokens.js`.
2. **No es el producto publicado** hasta que copies o refactorices el resultado dentro de `src/components/` (u otra ruta de producción) usando tokens y el mismo criterio que el resto del portfolio.
3. **Rutas de preview:** si más adelante montas una ruta solo para ver un mockup (React Router, otra entrada HTML, etc.), documenta en este README la **URL** y deja claro que **no** debe enlazarse desde el Navbar ni CTAs del sitio en producción.
4. **`mokups`** es el nombre de carpeta a propósito (convención del repo); el contenido igual es un mockup.

El portfolio principal no depende de nada en esta carpeta.
